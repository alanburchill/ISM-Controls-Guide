## Summary

Enable Edge policy to block intrusive ads by applying the Microsoft Edge security baseline in Intune[^1]. The baseline preconfigures Edge privacy settings to disable ad personalization and reduce tracking, supporting ad-blocking across devices[^2].

## Implementation Steps

### Enable Edge policy to block intrusive ads using Intune security baseline

1. Navigate to the Microsoft Intune admin center.

2. Select Endpoint security > Security baselines > Security Baseline for Microsoft Edge > Create policy > Create.

3. On the Basics tab:
   - Name: Microsoft Edge Security Baseline - Level 2
   - Description: Microsoft Edge security baseline providing Level 2 enhanced security

4. Select Next.

5. In Configuration settings, expand Microsoft Edge and review the preconfigured settings. Note: Privacy & Data > Allow personalization of ads, search, and news is Disabled. [^2]

6. Select Next.

7. For Scope tags, select the appropriate scope tag.

8. For Assignments, assign to SEB-Level2-Devices group.

9. Select Next to review the settings. Then choose Create.
